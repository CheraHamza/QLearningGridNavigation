# RLGridNavigation

This project is the front-end of the RLGridNavigation environment, which is a simple grid-based environment for a Q-learning agent. The environment consists of a grid where the agent can move in four directions (up, down, left, right) to reach a goal state while avoiding obstacles and checking checkpoints. The front end provides a user interface for visualizing and editing the environment and interacting with the agent's actions.

## Features

- Visual representation of the grid environment, including the agent, goal, obstacles, and checkpoints.
- Interactive editing of the environment, allowing users to add or remove obstacles and checkpoints.
- Real-time feedback on the agent's actions and the resulting state changes.
- Communication with the back end to process actions and update the environment state.
